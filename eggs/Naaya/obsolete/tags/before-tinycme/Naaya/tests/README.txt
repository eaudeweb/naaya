Usage:

1.download ZopeTestCase from http://www.zope.org/Members/shh/ZopeTestCase;

2.Extract the tarball into the lib/python/Testing directory of your Zope installation;

3.copy 'framework.py' from lib\python\Testing\ZopeTestCase directory of your Zope installation to this folder;

4.set up the environment variables in 'run_tests.bat' to reflect your system;

5.run 'run_tests.bat' and see the results.
